MAD-X current releases [Changelog]

  • The last MAD-X development version 5.03.06 was released the 2017.05.29.
  • The last MAD-X production version 5.03.06 was released the 2017.05.29.
  • New release numbering scheme changed after release 5.03.00 the 2017.04.15.
  • The 32 architectures and standalone libraries are provided only on demand since the 2016.04.06.

  • The libptc library was added to the development release 5.00.13 the 2012.03.09.
