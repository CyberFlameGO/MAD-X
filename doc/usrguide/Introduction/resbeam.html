RESBEAM: reset beam defaults

The RESBEAM command resets all beam data to their default values. 
RESBEAM, SEQUENCE=name;
The only argument to RESBEAM is the sequence name. If the sequence name is omitted, the default beam is reset.

Default BEAM Data: 

PARTICLE        POSITRON
ENERGY          1 GeV
EX              1 rad m
EY              1 rad m
ET              1 rad m
KBUNCH          1
NPART           0
BCURRENT        0 A
BUNCHED         .TRUE.
RADIATE         .FALSE.
